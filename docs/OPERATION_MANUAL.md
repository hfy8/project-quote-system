## 1. 系统概述

**项目报价系统** 是一个多角色协同的工业设备报价系统，支持模块化报价、人力/物料/差旅成本核算、跨币种汇率、多版本快照及 PDF 导出。

**核心概念**:
- **报价单 (Quotation)**: 顶层项目实体，如 "主板锁螺丝机"、"JOT流线组件"
- **模块 (Module)**: 报价单下的分类（机构 / 电气 / PLC / 其他），按 group 分组展示
- **物料 (Material)**: 模块下的具体物料条目（含型号、单价、用量、损耗率）
- **参与人员 (Participant)**: 报价单的协作者，按 `agency/electrical/project` 三种类型分配
- **版本 (Version)**: 报价单历史快照，可对比 / 回滚 / 导出 PDF

**技术栈**: Vue 3 + Element Plus + FastAPI + PostgreSQL，部署于 Docker Swarm (swarm01/swarm02)。

## 3. 通用流程

### 3.1 登录 / 登出

![login](manual_screenshots/01_admin_00_login.png)

**操作步骤**:

1. 打开浏览器，访问 `http://10.60.100.1:8080`
2. 自动跳转 `/login` 页，输入**用户名**和**密码**
3. 点击右下角 **登 录** 按钮（或按 Enter）
4. 登录成功后跳转 `/dashboard` 首页
5. 登录失败会显示红色错误提示："用户名或密码错误"

**登出**: 左下角用户卡片 → 红色 **退出登录** 按钮 → 回到登录页。

**注意**:
- 系统使用 JWT Token（7 天有效期），存储在 `localStorage.token`
- 关闭浏览器不会登出；切换账号前必须先登出
- 首次登录后建议修改默认密码

### 3.2 首页 Dashboard

![admin dashboard](manual_screenshots/01_admin_02_dashboard.png)

**操作步骤**:

1. 登录后自动进入首页
2. 顶部展示**欢迎语** + 当前角色徽章
3. 中部展示 **4 张 KPI 卡片**（报价单总数、待审批数、本月新增、参与报价单数）
4. 下部展示 **待办事项列表**（按角色不同）：
   - admin: 系统告警 + 全公司待审批
   - manager: 自己负责的报价单
   - sales: 自己作为 agency 参与的报价单
   - viewer: 自己作为 electrical 参与的报价单

**快捷入口**: 顶部 KPI 卡片可点击进入对应列表页。

### 3.3 报价单列表

![admin 报价单列表](manual_screenshots/01_admin_02_dashboard.png)

**操作步骤**:

1. 左侧导航 → **报价单管理**
2. 默认显示**所有可见报价单**（admin 看全部，其他角色看自己参与的）
3. 顶部筛选区：
   - **状态** 下拉：草稿/审批中/已审批/已归档/已驳回/已取消/审批中待定
   - **类型** 下拉：单项/线体/工装
   - **搜索框**：按名称、方案号模糊搜索
4. 表格列：项目名称、方案号、类型、状态、项目落地、创建人、操作
5. **状态彩色徽章**：
   - 草稿 (草稿) 灰色
   - 审批中 (审批中) 蓝色
   - 已审批 (已审批) 绿色
   - 已归档 (已归档) 紫色
   - 已驳回 (已驳回) 红色
   - 已取消 (已取消) 灰色
   - 审批中待定 (审批中待定) 橙色
6. 点击 **项目名称** 进入 view 页；点击 **+ 按钮** 创建新报价单

**注意**:
- admin 默认看不到自己不是 participant 的报价单的 tab 列表（必须在"参与人员" tab 加自己）
- admin 看列表用 `/quotations`，其他角色用 `/my-assignments` 进入"我的分配"

### 3.4 我的分配

![我的分配](manual_screenshots/01_admin_04_my_assignments.png)

**操作步骤**:

1. 左侧导航 → **我的分配**
2. 默认显示**所有状态**的报价单，可按状态筛选
3. 每行展示：项目名称、方案号、状态、分配时间、操作
4. **分配状态**（彩色徽章）：
   - 待开始 (info) 灰色
   - 进行中 (primary) 蓝色
   - 已完成 (success) 绿色
   - 已驳回 (danger) 红色
   - 已取消 (info) 灰色
   - 审批中 (warning) 橙色
   - 审批中待定 (warning) 橙色

**业务逻辑**:
- 此页面专门用于**跨报价单查看自己被分配的模块任务**
- 与报价单列表的区别：本页只显示**模块分配**（business_owner 创建的报价单中分给我的模块任务）

### 3.5 我的消息

![我的消息](manual_screenshots/01_admin_05_messages.png)

**操作步骤**:

1. 左侧导航 → **我的消息**

3. 未读消息**红色徽章**标在左侧导航

### 3.6 待审批归档

![待审批归档](manual_screenshots/01_admin_06_pending_approvals.png)

**操作步骤**:

## 4. 报价单 View — 按 Tab 详解

> 演示报价单: 主板锁螺丝机 (id=3)
> URL: `/quotations/3/view`
> 顶部标题: "编辑报价单 - 主板锁螺丝机"

### 4.1 基本信息

![基本信息](manual_screenshots/01_admin_20_view_01_basic.png)

**Tab 顺序**: 第 1 个，**所有角色都可见**。

**操作步骤**:

1. 点击顶部 **基本信息** tab
2. 看到表单字段：
   - **项目名称**（必填）
   - **方案号**（必填，唯一）
   - **类型**: 单项 / 线体 / 工装
   - **状态**: 草稿 / 审批中 / 已审批 / 已归档
   - **项目落地**: 客户名称 + 地点
   - **客户联系人** + **联系方式**
   - **创建人**（自动填充）
   - **创建时间**（自动填充）
3. 修改字段 → 点右下角 **保存** 按钮 → 成功提示 "保存成功"

**注意**:
- 状态变为"已审批"后不能再编辑（需先撤销归档）
- 草稿状态可任意修改
- 方案号重复会报错 "方案号已存在"

### 4.2 模块管理

![模块管理](manual_screenshots/01_admin_20_view_02_modules.png)

**Tab 顺序**: 第 2 个，**所有角色都可见**。

**操作步骤**:

1. 点击 **模块管理** tab
2. 看到 **4 个分组卡片**（按 module.type 分组）：
   - 🔵 **机构模块** (agency type)
   - 🟢 **电气模块** (electrical type)
   - ⚡ **PLC 模块** (project type)
   - 📦 **其他模块** (other)
3. 每个分组卡片显示该类型下模块数量徽章
4. 模块列表项包含：模块名、模块编号、描述
5. **新建模块**: 右上角 ➕ → 弹窗填写 → 选择所属分组 → 保存
6. **删除模块**: 模块右侧 🗑️ 红色按钮 → 确认弹窗 → 删除

**跨报价单复制模块**:
1. 点击 **从其他报价单复制模块** 按钮（在 Tab 右上方）
2. 弹窗显示分页 table，列出**所有可见报价单的所有模块**（不限人员）
3. 顶部搜索框可按模块名 / 报价单名搜索
4. 分页条支持 10/20/50 条/页
5. 单击某行选中 → **复制选中模块** 按钮 → 自动追加到当前报价单

**注意**:
- 删除模块会**级联删除**该模块下的所有物料
- 复制模块时只复制**模块结构**，不复制物料数据
- 已审批的报价单不能新建/删除模块

### 4.3 物料清单

![物料清单](manual_screenshots/01_admin_20_view_06_materials.png)

**Tab 顺序**: 第 6 个，**sales / viewer / admin 可见**，**manager 不可见**。

**操作步骤**:

1. 点击 **物料清单** tab
2. 看到 **模块树** + **物料表** 双栏布局：
   - 左侧树状图：选中模块 → 右侧显示该模块的物料
3. 物料表格列：序号、物料名、规格型号、单位、用量、损耗率、单价、含系数价、合计
4. **新增物料**: 右上 ➕ → 弹窗选择原材料库 → 自动填充单价 → 输入用量/损耗率
5. **批量导入**: 上传 Excel 模板 → 解析 → 预览 → 确认
6. **导出 Excel**: 右上角 ↓ 按钮

**聚合视图**:
- 顶部 **按汇总聚合** 切换：相同物料（型号 + 规格）合并显示
- "数量+2" 模式：按机构/电气/PLC 三个分组分别显示
- 默认按模块 + 物料分组

**注意**:
- 物料的单价 × 用量 × (1 + 损耗率) = 含系数价
- "其他成本锁死"假设：物料不变时，修改系数只调整其他费用部分
- "数量+2" 模式下，删除物料必须按 mm.id（ModuleMaterial 主键）而非 material_id，否则会误合并

### 4.11 汇总

![汇总](manual_screenshots/01_admin_20_view_12_summary.png)

**Tab 顺序**: 第 12 个，**admin / manager / sales 可见**，**viewer 不可见**。

**操作步骤**:

1. 点击 **汇总** tab
2. 顶部 **6 张分组卡片**（永远显示全部分组，无数据时为 0 + "暂无数据"）：
   - 物料成本（含系数）
   - 人工成本
   - 差旅成本
   - 运输包装
   - 费用系数
   - **总价**
3. 下方 **分模块明细 table**：模块名、各分类金额、模块合计
4. **导出汇总 PDF**: 右上角按钮 → 浏览器自动下载 PDF
5. **目标价反推**: 顶部"目标价"输入框 → 输入目标金额 → **反推系数** → 系统自动建议各分组调整

**注意**:
- 汇总页面所有**百分比**精确到小数点后两位 (`(x*100).toFixed(2)%`)
- 物料差异 key 用 mm.id (ModuleMaterial 主键) 而非 material_id
- "其他成本锁死"假设：物料不变时修改系数只调整其他费用

## 6. 常见问题 FAQ

### Q1: 报价单列表是空的？

A: 你的账号**没有被加入任何报价单的参与人员**。请 admin 在该报价单的"参与人员" tab 添加你，选择合适的类型（agency/electrical/project）。

### Q2: 看不到某个 Tab？

A: 该 Tab 对你当前的参与类型不可见。参考 §2.2 权限矩阵。例如业务员（agency）看不到"参与人员"/"费用"/"差旅" tab。

### Q3: 状态已审批后还能修改吗？

A: 不能。需先在"待审批归档"页 **驳回** 或 **撤销归档** 回到草稿状态。

### Q4: 删除模块会丢失物料吗？

A: **会**。删除模块会级联删除该模块下的所有物料和 ModuleMaterial 关联。删除前建议先创建版本快照。

### Q5: 汇总页百分比显示不对？

A: 汇总页所有百分比精确到小数点后两位（`toFixed(2)`），默认显示 `0.00%`。如果显示为整数，检查浏览器缓存（Ctrl+F5）。

### Q6: 导出 PDF 报 `Cannot read properties of null (reading 'querySelectorAll')`？

A: 这是已修复的 bug（v1.3.77+）。原因：拆组件后 `summaryRef` 指向子组件实例而非 DOM。修复：使用 `summaryRef.value?.summaryRef` 获取真实 DOM。请升级到 frontend v1.3.77+。

### Q7: 分配报价单状态不全？

A: 状态枚举包括：待开始 / 进行中 / 已完成 / 已驳回 / 已取消 / 审批中 / **审批中待定**。如果某个状态没显示，请升级到 frontend v1.3.78+（该版本补全了 `approved_pending` 映射）。

### Q8: AI 助手没有响应？

A: 检查后端 LLM 服务是否在线。AI 助手页面右下角有状态指示器。离线模式下返回 fallback 响应。

### Q9: 跨报价单复制模块看不到其他报价单？

A: 升级到 backend v1.4.36+（已添加 `GET /api/quotations/all-modules` 接口）。admin 看全部，普通用户按参与过滤。

### Q10: 物料清单 tab 显示但看不到？

A: 即使有 view 权限，没有 edit 权限时该 tab 可能仍隐藏。检查 `permissions.tabs` 是否包含 'materials'。
